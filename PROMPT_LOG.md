Модель репетитора содержит:
- id (int, автогенерация)

- GET /tutors/{id} — получить по ID
- PUT /tutors/{id} — обновить (частичное обновление)
- DELETE /tutors/{id} — удалить

Используй Pydantic v2 для валидации. Добавь обработку ошибок (404, 422).
Используй in-memory хранилище (dict). Покрой код type hints и напиши краткую документацию.
